koc-api
=======

Web API for the online MMORPG Kings of Chaos.<br>
A demo is running at http://koc-api.herokuapp.com/api/ (not warranty, might be down/broken).

Usage
-----

### Register
**Path:**   /api/register<br>
**Method:** POST<br>
**Data:**<br>
  username<br>
  password<br>
  email<br>
  race<br>
  challenge<br>
  challenge_response<br>
